#### MERN CMS

a practice portfolio project
by josh kaye


### Project Structure

Pages:
- Index : show posts
- View Single Post
- Admin : manage posts
  - View all posts
  - edit single post
  - delete post

API Routes:

- GET  /api/ -> show all posts
- POST /api/ -> create new post

- GET    /api/post/:slug -> show single post
- PUT    /api/post/:id -> update single post
- DELETE /api/post/:id -> destroy single post
