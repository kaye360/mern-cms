#### MERN CMS

a practice portfolio project
by josh kaye


### Project Structure

Pages:
- Index : show posts
- View Single Post
- Admin : manage posts
  - View all posts
  - edit single post
  - delete post

API Routes:

- GET  / - show all posts
- POST / - create new post

- GET    /post/:id show single post
- PUT    /post/:id update single post
- DELETE /post/:id destroy single post
